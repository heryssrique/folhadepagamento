# FolhaPay – Verificação & Validação da Folha

Aplicação para conferência, auditoria e fechamento de folha de pagamento.

## ✨ Funcionalidades
- Upload de Planilhas (XLSX, CSV) e Relatórios PDF.
- Validação automática de CPFs, Salário Mínimo e Encargos.
- Dashboard com KPIs em tempo real.
- Workflow de fechamento de competência.
- Geração de relatórios consolidados.

## 🚀 Como Executar Localmente
1. Baixe os arquivos.
2. Abra o `index.html` ou use um servidor estático:
   ```bash
   npm install
   npm start
   ```

## ☁️ Deploy no Vercel
Esta aplicação está pronta para ser publicada no Vercel. 
Basta conectar o repositório no dashboard da Vercel ou usar o CLI:
```bash
vercel
```
